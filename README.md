# Colliding Blocks  

### Compute the digits of pi тлк by counting the number of collisions of two blocks  

Check **3Blue1Brow** - [video](https://www.youtube.com/watch?v=HEfHFsfGXjs) for the explanation of why do colliding blocks compute тлк.  

To use the code you need fisrt to install **PyGame**.  
* for windows users :  
- `pip install pygame`  

* For Mac/linux users :
- `pip3 install pygame`
